# Table Leg Reference Prompt

## Purpose
This prompt is used to specify the table leg style and construction that will be combined with the tabletop surface.

## Prompt Template

```
Generate or describe table legs with the following specifications:

Leg Design:
- Style: [LEG_STYLE - e.g., X-style, Diamond, Four-Y, Sandtimer, Square, Pear-half]
- Material: Black powder-coated steel
- Construction: [e.g., Welded steel frame, Bolt assembly, etc.]
- Dimensions: [HEIGHT]mm height, [SPECIFICATIONS for cross-sections]
- Weight capacity: [CAPACITY in kg/lbs]

Visual characteristics:
- Finish: Matte black powder coating
- Texture: Smooth industrial finish
- Details: [Clean welds, Minimal joints, Modern aesthetic, etc.]
- Mounting: [Top plate design, Attachment method]

The leg design should:
- Complement modern and industrial interior styles
- Provide stable support for the tabletop
- Feature clean lines and minimal aesthetic
- Show professional craftsmanship in the metal work
```
